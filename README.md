# Tarea 4 - Prueba Técnica desarrollo de API Rest

---

## 📌 Descripción

Este proyecto corresponde a la **Tarea 4**, cuyo objetivo es desarrollar una aplicación web **full-stack** que permita comprender la integración entre:

* Backend (API REST)
* Frontend
* Base de datos

A través de este ejercicio se implementa un flujo básico de autenticación (**login**) y la comunicación entre las distintas capas del sistema mediante solicitudes HTTP.

---

## 🏗️ Tecnologías utilizadas

* **Backend:** Spring Boot
* **Frontend:** (Vue.js / React / Angular)
* **Base de datos:** PostgreSQL


---

## 🎯 Objetivos

* Diseñar y consumir una API REST
* Implementar autenticación básica
* Gestionar modelos de datos y migraciones
* Comprender la arquitectura full-stack

---

## 📂 Estructura del proyecto

El proyecto sigue una arquitectura cliente-servidor:

```
/backend
/frontend
```

---

## 🚀 Notas

Este proyecto sirve como base (**scaffolding**) para el desarrollo del proyecto final del curso, permitiendo extender funcionalidades y adaptar el modelo de datos según las necesidades.
